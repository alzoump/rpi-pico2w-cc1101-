# rpi-pico2w-cc1101

Spectrum Analyzer, Capture & Replay, Protocol Sniffer, and Radio Information Tool using Raspberry Pi Pico 2W and CC1101.

---

## 🔧 Hardware Required
- Raspberry Pi Pico 2W
- CC1101 wireless transceiver module
- Jumper wires

## ⚡ Wiring

| CC1101 Pin | Pico 2W Pin |
|------------|-------------|
| VCC        | 3.3V        |
| GND        | GND         |
| MOSI       | GP19        |
| MISO       | GP16        |
| SCK        | GP18        |
| CSN        | GP17        |
| GDO0       | GP20        |
